About

I am a machine learning and data science professional with experience building models and analytics for complex operational and customer-facing settings. My work includes customer behavior modeling, sentiment analysis using transformer-based NLP models, time series forecasting, dimensionality reduction (PCA, t-SNE), and quantitative segmentation.

I originally trained in business and management and hold two MBAs. My master's research examined structural barriers to career advancement for women managers through survey-based empirical analysis. That work introduced me to system-level thinking, feedback loops, and how metrics and structures shape outcomes—ideas that continue to influence how I approach machine learning and data systems today.

In recent years I have completed applied data science and cloud training through MIT Professional Education and Great Learning, and I am building a portfolio that bridges hands-on ML work with broader questions about robustness, governance, and decision impact.

Throughout my career, I have mentored junior analysts on ML modeling, drift detection, and evaluation frameworks, and led internal knowledge sessions on bias detection and model validation. I believe effective mentorship involves helping others build intuition about why methods work, not just how to apply them.

Representation & Modeling

  • SVD-Based Recommendation Systems (2024-Present)
    Python • Surprise Library • SVD • Collaborative Filtering

    Implemented SVD-based collaborative filtering and rank-based recommenders on user–item interaction data. Analyzing how data sparsity and long-tail distributions distort embedding geometry in matrix factorization models, where rare users and items exhibit unstable latent representations under small perturbations. Helped build intuition about how different representations and data regimes influence model behavior and reliability.

    View on GitHub →
  • CNN vs. ANN for Digit Recognition (2024)
    TensorFlow • CNN • ANN • MNIST

    Built and compared convolutional and fully connected networks for digit classification, analyzing generalization and robustness under different training regimes and architecture depths. Examining how architectural inductive biases shape learned feature geometry.

    View on GitHub →
  • Dimensionality Reduction & Latent Structure (2024)
    PCA • t-SNE • clustering • visualization

    Applied PCA and t-SNE to high-dimensional data, interpreting eigenstructure, variance explained, and cluster separation via silhouette scores and visual inspection. Exploring how dimensionality reduction reveals or obscures underlying data structure.

    View on GitHub →

NLP, Imbalanced Data & Pipelines

  • Sentiment Analysis with Transformer Models (2024-Present)
    Hugging Face Transformers • NLP • Sentiment • Error Analysis

    Built sentiment classifiers on customer text using transformer-based models. Conducting systematic error analysis to examine how representation behavior correlates with model failures under ambiguity and domain shift. Focused on preprocessing, fine-tuning, and evaluation using accuracy, F1, and confusion matrices with targeted error analysis.

    View on GitHub →
  • Predictive Modeling Under Class Imbalance (2024-Present)
    Python • Imbalanced data • ROC-AUC • F1

    Developed models for imbalanced prediction problems (e.g., approval risk, cancellation risk) using resampling strategies and threshold tuning. Developing diagnostic frameworks that go beyond aggregate accuracy to surface performance degradation in minority classes. Evaluated performance via ROC-AUC, precision–recall, and F1 scores across different class distributions.

    View on GitHub →
