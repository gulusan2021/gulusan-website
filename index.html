Representation & Model Behavior

  • Representation Geometry in SVD-Based Recommendation Systems (2024–present)
    Python • SVD • Collaborative Filtering • Representation Analysis

    Analyzes how data sparsity and long-tail user–item distributions distort embedding geometry in SVD-based collaborative filtering models. Examines instability of latent representations for rare users and items, highlighting how small perturbations in sparse regimes lead to large representational shifts that are not captured by aggregate accuracy metrics.

    View on GitHub →
  • CNN vs. ANN: Representation Behavior in Image Classification (2024)
    TensorFlow • CNN • ANN • Inductive Bias

    Compares convolutional and fully connected neural networks on image classification tasks, focusing on differences in learned representations and generalization behavior. Investigates how architectural inductive biases shape feature geometry, robustness, and performance under varying training conditions.

    View on GitHub →
  • Latent Structure Analysis with PCA and t-SNE (2024)
    PCA • t-SNE • Clustering • Diagnostic Analysis

    Explores latent structure in high-dimensional datasets using PCA and t-SNE, combined with clustering diagnostics and variance analysis. Examines when dimensionality reduction reveals meaningful structure versus when it obscures relationships, emphasizing interpretability and diagnostic use rather than visualization alone.

    View on GitHub →

Evaluation, NLP & System Diagnostics

  • Transformer Error Analysis Under Distribution Shift (2024–present)
    Hugging Face • NLP • Error Analysis • Distribution Shift

    Conducts systematic error analysis of transformer-based sentiment classifiers under linguistic ambiguity and distribution shift. Links model failures to representational uncertainty, highlighting how confident predictions can mask systematic errors that standard evaluation metrics fail to surface.

    View on GitHub →
  • Diagnostics for Class Imbalance in Predictive Models (2024–present)
    Imbalanced Data • ROC • Precision–Recall • Threshold Analysis

    Develops diagnostic frameworks for predictive modeling under severe class imbalance using ROC curves, precision–recall analysis, and threshold tuning. Demonstrates how aggregate accuracy can obscure minority-class failures and proposes evaluation strategies that better reflect operational risk.

    View on GitHub →
